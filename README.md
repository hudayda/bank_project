# Bank Management System (Python + HTML/CSS)

## Project Overview
A simple banking management system built with **Python** for the backend logic and **HTML/CSS** for the user interface.  
The project focuses on practicing **clean code**, basic **financial operations**, and building a clear, user-friendly layout.

## Features
- Create a user/account
- View account details and current balance
- Deposit money
- Withdraw money (with basic validation)
- Simple transaction flow (UI pages or sections)

> Note: This is a learning project; it is not intended for real financial use.

## Tech Stack
- **Python** (backend logic)
- **HTML** (structure)
- **CSS** (styling)
